# SPI CONTROLLER

I have made a custom SPI master and slave that work in the mode 1 of SPI
This folder contains 2 parts
1. SPI Master and Slave along with controller to control through the serial terminal, implemented in a single FPGA fabric on a single board
2. SPI Master and Slave along with a master controller and a slave controller through the serial terminal of their own, implemented on two seperate boards on the FPGA fabric of their own and their own processing system
