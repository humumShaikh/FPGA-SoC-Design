In this project, we will look into how we can do a simple integration between the Zynq Processing system and the Programmable Logic.
We will implement a NAND gate.
The Programmable Logic (PL) will have the AND Gate.
The Processing System (PS) will negate the output of the AND gate received from the PL.
The final output will then be output through the PS.
The inputs to the logic will be given through the PL.
